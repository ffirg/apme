description: Use failed_when or register instead of ignore_errors.

## Ignore errors

Use failed_when or register instead of ignore_errors.

### Example: violation

```yaml
- name: Risky task
  ansible.builtin.command: whoami
  ignore_errors: true
```

### Example: pass

```yaml
- name: Safe task
  ansible.builtin.command: whoami
  ignore_errors: true
  register: result
```
