## Package loop (L069)

Package modules accept lists natively. Batch package names in the `name:`
parameter instead of looping with `item`.

### Example: violation

```yaml
- name: Install packages
  ansible.builtin.apt:
    name: "{{ item }}"
    state: present
  loop:
    - httpd
    - nginx
```

### Example: pass

```yaml
- name: Install packages
  ansible.builtin.apt:
    name:
      - httpd
      - nginx
    state: present
```
