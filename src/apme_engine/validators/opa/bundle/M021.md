## Empty args keyword (M021)

Empty args: keyword on a task is deprecated; remove it (2.23)

**Removal version**: 2.23
**Fix tier**: 1
**Audience**: content

### Detection

Detect args: with null value in tasks

### Example: violation

```yaml
- name: Run command
  ansible.builtin.command: whoami
  args:
```

### Example: pass

```yaml
- name: Run command
  ansible.builtin.command: whoami
```

### Remediation

Remove the empty args: key
