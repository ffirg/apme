## No changed_when

command/shell/raw need changed_when or creates/removes.

### Example: violation

```yaml
- name: Echo
  ansible.builtin.command:
    cmd: echo hello
```

### Example: pass

```yaml
- name: Echo
  ansible.builtin.command:
    cmd: echo hello
  changed_when: false
```
