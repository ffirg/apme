## Empty when conditional (M016)

Empty when: conditional is deprecated; remove it or add an explicit condition (2.23)

**Removal version**: 2.23
**Fix tier**: 1
**Audience**: content

### Detection

Detect when: with empty string or null value in tasks

### Example: violation

```yaml
- name: Always runs
  ansible.builtin.debug:
    msg: "This always runs"
  when: ""
```

### Example: pass

```yaml
- name: Always runs
  ansible.builtin.debug:
    msg: "This always runs"
```

### Remediation

Remove the empty when: key
