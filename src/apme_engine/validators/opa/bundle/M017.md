## action as mapping (M017)

action: with a mapping value is deprecated; use string form or module key directly (2.23)

**Removal version**: 2.23
**Fix tier**: 2
**Audience**: content

### Detection

Detect action: key with dict/mapping value

### Example: violation

```yaml
- name: Copy file
  action:
    module: copy
    src: a
    dest: /tmp/b
```

### Example: pass

```yaml
- name: Copy file
  ansible.builtin.copy:
    src: a
    dest: /tmp/b
```

### Remediation

Convert action mapping to module key with args
