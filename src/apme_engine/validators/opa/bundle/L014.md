## No handler

Use notify/handler instead of when: result.changed.

### Example: violation

```yaml
- name: Conditional on change
  ansible.builtin.debug:
    msg: something changed
  when: result.changed
```

### Example: pass

```yaml
- name: Simple task
  ansible.builtin.debug:
    msg: hello
```
