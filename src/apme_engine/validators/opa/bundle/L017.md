## No relative paths

Avoid relative path in src.

### Example: violation

```yaml
- name: Copy file
  ansible.builtin.copy:
    src: ../other/file.txt
    dest: /tmp/file.txt
```

### Example: pass

```yaml
- name: Copy file
  ansible.builtin.copy:
    src: files/config.txt
    dest: /tmp/config.txt
```
