## Latest

Avoid state=latest; pin package versions.

### Example: violation

```yaml
- name: Install package
  ansible.builtin.apt:
    name: nginx
    state: latest
```

### Example: pass

```yaml
- name: Install package
  ansible.builtin.apt:
    name: nginx
    state: present
```
