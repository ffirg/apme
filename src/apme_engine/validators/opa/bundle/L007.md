## Command instead of shell

Prefer command when no shell features needed.

### Example: violation

```yaml
- name: Echo
  ansible.builtin.shell: echo hello
```

### Example: pass

```yaml
- name: Grep with pipe
  ansible.builtin.shell: cat file | grep foo
```
