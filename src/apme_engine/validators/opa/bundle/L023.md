## Run once

Consider whether run_once is appropriate.

### Example: violation

```yaml
- name: Run once task
  ansible.builtin.debug:
    msg: hi
  run_once: true
```

### Example: pass

```yaml
- name: Normal task
  ansible.builtin.debug:
    msg: hi
```
