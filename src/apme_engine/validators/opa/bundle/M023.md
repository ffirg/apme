## follow_redirects string boolean (M023)

follow_redirects: yes/no (string) is deprecated in url lookup; use true/false boolean (2.22)

**Removal version**: 2.22
**Fix tier**: 1
**Audience**: content

### Detection

Detect follow_redirects: yes or follow_redirects: no in lookup args

### Example: violation

```yaml
- name: Fetch URL
  ansible.builtin.debug:
    msg: '{{ lookup("url", "https://example.com", follow_redirects="yes") }}'
```

### Example: pass

```yaml
- name: Fetch URL
  ansible.builtin.debug:
    msg: '{{ lookup("url", "https://example.com", follow_redirects=true) }}'
```

### Remediation

String to boolean substitution
