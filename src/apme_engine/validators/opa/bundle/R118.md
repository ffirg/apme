## Inbound transfer (R118)

Detects tasks that perform an inbound network transfer from an external source (annotation-based). Fires when the ARI engine annotates a task with `risk_type: inbound_transfer`, which covers modules like `get_url`, `uri`, `unarchive` with a remote source, etc.

### Example: violation

```yaml
- name: Download artifact
  ansible.builtin.get_url:
    url: https://example.com/file.tar.gz
    dest: /tmp/file.tar.gz
```

### Example: pass

```yaml
- name: No download
  ansible.builtin.debug:
    msg: hello
```
