---
rule_id: L020
validator: opa
description: mode should be string with leading zero.
---

## Risky octal

mode should be string with leading zero.

### Example: violation

```yaml
- name: Copy file
  ansible.builtin.copy:
    src: foo
    dest: /tmp/foo
    mode: 644
```

### Example: pass

```yaml
- name: Copy file
  ansible.builtin.copy:
    src: foo
    dest: /tmp/foo
    mode: "0644"
```
