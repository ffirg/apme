rule_id: L015
validator: opa
description: Avoid Jinja in when; use variables.
---

## No Jinja in when

Avoid Jinja in when; use variables.

### Example: violation

```yaml
- name: Conditional task
  ansible.builtin.debug:
    msg: hi
  when: "{{ my_var }}"
```

### Example: pass

```yaml
- name: Conditional task
  ansible.builtin.debug:
    msg: hi
  when: my_var
```
