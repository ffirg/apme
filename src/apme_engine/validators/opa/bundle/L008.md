---

## No local_action

Use delegate_to: localhost instead of local_action.

### Example: violation

```yaml
- name: Run locally
  local_action: command whoami
```

### Example: pass

```yaml
- name: Run locally
  ansible.builtin.command: whoami
  delegate_to: localhost
```
