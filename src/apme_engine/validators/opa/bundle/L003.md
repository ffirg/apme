## Play should have name

Each play should have a name.

### Example: violation

```yaml
- hosts: localhost
  tasks:
    - name: Do something
      ansible.builtin.debug:
        msg: hello
```

### Example: pass

```yaml
- name: My play
  hosts: localhost
  tasks:
    - name: Do something
      ansible.builtin.debug:
        msg: hello
```
