---
rule_id: L005
validator: opa
description: Use ansible.builtin or ansible.legacy.
---

## Only builtins

Use ansible.builtin or ansible.legacy.

### Example: violation

```yaml
- name: Network CLI
  ansible.netcommon.network_cli:
    connection: network_cli
```

### Example: pass

```yaml
- name: Copy file
  ansible.builtin.copy:
    src: foo
    dest: /tmp/foo
```
