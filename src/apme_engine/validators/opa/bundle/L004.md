## Deprecated module

Do not use deprecated modules.

### Example: violation

```yaml
- name: Include tasks
  include: tasks/setup.yml
```

### Example: pass

```yaml
- name: Copy file
  ansible.builtin.copy:
    src: foo
    dest: /tmp/foo
```
