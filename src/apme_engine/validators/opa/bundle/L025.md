validator: opa
description: Task/play name should start with uppercase.
---

## Name casing

Task/play name should start with uppercase.

### Example: violation

```yaml
- name: my play
  hosts: localhost
  tasks:
    - name: do something
      ansible.builtin.debug:
        msg: hi
```

### Example: pass

```yaml
- name: My play
  hosts: localhost
  tasks:
    - name: Do something
      ansible.builtin.debug:
        msg: hi
```
