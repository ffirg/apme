rule_id: L018
validator: opa
description: become_user should have corresponding become.
---

## Partial become

become_user should have corresponding become.

### Example: violation

```yaml
- name: Run as root
  ansible.builtin.command: whoami
  become_user: root
```

### Example: pass

```yaml
- name: Run as root
  ansible.builtin.command: whoami
  become: true
  become_user: root
```
