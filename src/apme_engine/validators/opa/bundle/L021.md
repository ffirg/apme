## Risky file permissions

Set mode explicitly for file/copy/template.

### Example: violation

```yaml
- name: Copy file
  ansible.builtin.copy:
    src: foo
    dest: /tmp/foo
```

### Example: pass

```yaml
- name: Copy file
  ansible.builtin.copy:
    src: foo
    dest: /tmp/foo
    mode: "0644"
```
