---

## Literal compare

Avoid literal true/false in when.

### Example: violation

```yaml
- name: Conditional task
  ansible.builtin.debug:
    msg: hi
  when: my_var == true
```

### Example: pass

```yaml
- name: Conditional task
  ansible.builtin.debug:
    msg: hi
  when: my_var
```
