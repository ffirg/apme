description: Avoid comparison to empty string in when.
---

## Empty string compare

Avoid comparison to empty string in when.

### Example: violation

```yaml
- name: Conditional task
  ansible.builtin.debug:
    msg: hi
  when: my_var == ""
```

### Example: pass

```yaml
- name: Conditional task
  ansible.builtin.debug:
    msg: hi
  when: not my_var
```
