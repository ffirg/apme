## Command instead of module

Use dedicated module instead of command.

### Example: violation

```yaml
- name: Install package
  ansible.builtin.command: apt-get install nginx
```

### Example: pass

```yaml
- name: Install package
  ansible.builtin.apt:
    name: nginx
    state: present
```
