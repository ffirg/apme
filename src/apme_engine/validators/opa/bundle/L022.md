rule_id: L022

description: Shell with pipe should set set -o pipefail.
---

## Risky shell pipe

Shell with pipe should set set -o pipefail.

### Example: violation

```yaml
- name: Grep with pipe
  ansible.builtin.shell: cat file | grep foo
```

### Example: pass

```yaml
- name: Grep with pipe
  ansible.builtin.shell: set -o pipefail; cat file | grep foo
```
